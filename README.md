# 📊 Sales Dashboard Analysis

## 📌 Descripción
Este proyecto consiste en el desarrollo de un dashboard interactivo en Power BI para analizar el comportamiento de ventas, identificar patrones de ingresos y evaluar la rentabilidad por categoría, región y producto.

---

## 🎯 Objetivo
Analizar los datos de ventas con el fin de detectar oportunidades de mejora en el rendimiento del negocio y apoyar la toma de decisiones basada en datos.

---

## 🛠️ Herramientas utilizadas
- Power BI  
- Microsoft Excel

## ⚙️ Proceso

### 1. Limpieza de datos
- Eliminación de registros vacíos o inconsistentes  
- Validación de formatos (fechas, valores numéricos)  
- Preparación del dataset para análisis  

### 2. Modelado de datos
- Definición de métricas clave (ventas, profit, número de órdenes)  
- Organización de dimensiones (categoría, región, tiempo)  

### 3. Visualización
- Creación de dashboard interactivo en Power BI  
- Implementación de filtros por categoría, región y fecha  
- Diseño enfocado en claridad y facilidad de interpretación

## 🔍 Insights principales

- La categoría con mayor volumen de ventas no es la más rentable, lo que sugiere oportunidades de optimización en costos o precios.  
- Se identifica un comportamiento estacional en las ventas, con picos en determinados meses del año.  
- Algunas regiones presentan alto volumen de ventas pero bajos márgenes de ganancia, indicando posibles ineficiencias operativas.  

---

## 📊 Conclusión
El análisis permitió identificar patrones clave en el comportamiento de ventas y áreas de mejora en rentabilidad. Este tipo de dashboards facilita la toma de decisiones estratégicas basadas en datos reales.
